1. Instalar miniconda: https://docs.conda.io/en/latest/miniconda.html
2. Abrir la conosola de miniconda miniconda: windows -> anaconda
3. Ubicarse en el folder del repositorio (copiar y pegar la direcciòn exacta en el que se guardò del repositorio) 
	Ejemplo:
	>>> cd C:\Users\User1\Desktop\MAAD_GUI 

4. Desde la consola instalar las dependencias corriendo la liena:
	>>> conda env create -f environment.yml
	en caso de que el ambiente ya estè instalado y se desee actualizar correr la linea:
	>>> conda env update -f environment.yml --prune

4. Activar el ambiente
	>>>conda activate maad_gui

5. Ejecutar el programa en python
	>>> python maad_gui.py
